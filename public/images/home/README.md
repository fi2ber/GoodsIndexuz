# Требования к изображениям для главной страницы

## Структура папок

```
public/images/home/
├── hero/
│   └── hero-background.jpg
├── categories/
│   ├── nuts.jpg
│   ├── dried-fruits.jpg
│   ├── legumes.jpg
│   └── default.jpg (fallback)
├── origin/
│   ├── field.jpg
│   ├── processing.jpg
│   └── landscape.jpg
└── certificates/ (опционально)
    ├── iso.png
    ├── haccp.png
    └── organic.png
```

## Детальные требования

### 1. Hero Background (1 изображение)

**Путь**: `hero/hero-background.jpg`

- **Формат**: JPG или WebP
- **Разрешение**: 
  - Стандарт: 1920x1080px (16:9)
  - Retina: 3840x2160px (опционально)
- **Размер файла**: Максимум 1MB
- **Содержание**: 
  - Поле с сухофруктами/орехами
  - Склад с продуктами в упаковке
  - Производственный процесс
  - Профессиональная съемка, хорошее освещение
  - B2B стиль (не consumer)
- **Стиль**: Минималистичный, чистый, профессиональный

### 2. Категории продуктов (6-9 изображений)

**Путь**: `categories/[category-slug].jpg`

- **Формат**: JPG или WebP
- **Разрешение**: 
  - Стандарт: 800x600px (4:3)
  - Retina: 1600x1200px (опционально)
- **Размер файла**: Максимум 500KB каждое
- **Содержание** (по категориям):
  - **nuts.jpg** - Микс орехов (миндаль, фисташки, кешью, грецкий орех) на белом/светлом фоне
  - **dried-fruits.jpg** - Изюм, курага, чернослив в профессиональной упаковке или на светлом фоне
  - **legumes.jpg** - Фасоль, маш на светлом фоне
  - Добавьте изображения для других категорий по мере необходимости
- **Стиль**: 
  - Минималистичный
  - Чистый белый/светлый фон
  - Профессиональная съемка
  - Хорошее освещение
  - Продукты должны быть четко видны

### 3. О происхождении (2-3 изображения)

**Путь**: `origin/[image-name].jpg`

- **Формат**: JPG или WebP
- **Разрешение**: 
  - Стандарт: 1200x800px (3:2)
  - Retina: 2400x1600px (опционально)
- **Размер файла**: Максимум 800KB каждое
- **Содержание**:
  - **field.jpg** - Поле/плантация с продуктами (широкий план)
  - **processing.jpg** - Процесс сбора/обработки продуктов (средний план)
  - **landscape.jpg** - Регион/ландшафт Узбекистана (широкий план)
- **Стиль**: 
  - Профессиональная съемка
  - Естественное освещение
  - Показывает регион и процесс производства

### 4. Логотипы сертификатов (опционально, 2-4 изображения)

**Путь**: `certificates/[certificate-name].png`

- **Формат**: PNG с прозрачным фоном или SVG
- **Разрешение**: 200x100px (или пропорционально)
- **Размер файла**: Максимум 50KB каждое
- **Содержание**: 
  - ISO сертификат
  - HACCP сертификат
  - Organic сертификат
  - Другие релевантные сертификаты
- **Стиль**: 
  - Логотипы сертификатов
  - Прозрачный фон
  - Высокое качество

## Технические требования ко всем изображениям

1. **Качество**: Высокое (80-90% для JPG)
2. **Оптимизация**: Все изображения должны быть оптимизированы для веба
3. **Цветокоррекция**: Естественные цвета, без пересветов/недосветов
4. **Ориентация**: Горизонтальная (landscape) для всех изображений
5. **Фокус**: Резкие, четкие изображения
6. **Композиция**: Профессиональная, с учетом правил композиции

## Рекомендации по съемке

1. **Освещение**: Естественный свет или профессиональное студийное освещение
2. **Фон**: Чистый, минималистичный (белый, светло-серый, натуральный)
3. **Угол съемки**: Профессиональные ракурсы, показывающие качество продуктов
4. **Стиль**: B2B, не consumer - серьезный, профессиональный подход
5. **Детали**: Продукты должны быть четко видны, без размытия

## Примечания

- Если изображение отсутствует, сайт будет использовать градиентный фон или placeholder
- Все изображения должны быть оптимизированы перед загрузкой
- Рекомендуется использовать WebP формат для лучшей оптимизации
- Для retina дисплеев можно подготовить версии в 2x разрешении
